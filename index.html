Moderation

k!mute - Delete write permissions

This command allow to you mute/no allow writte someone person. k!mute @user

k!warn - Give a warn to someone

Warn someone, a certain number of warnings will mean an automatic warning action. k!warn @user < reason >

k!ban - Ban user from the server

This allows you to ban a user permanently, preventing them from entering the server until k!unban is used to allow their entry again k!ban @user < reason >
